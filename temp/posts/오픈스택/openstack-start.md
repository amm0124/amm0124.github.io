# OpenStack

OpenStack이란 `클라우드를 위한 오픈 소스 클라우드 컴퓨팅 플랫폼`입니다.


# OpenStack 아키텍처

- Keystone : 인증 제공자
- Ceilometer : 모니터링
- Horizon : UI 제공자

OpenStack은 여러 독립적인 부분인 OpenStack으로 이루어짐.
서비스는 서버라고 이해하면 될 듯.
