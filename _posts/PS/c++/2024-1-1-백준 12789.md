# 백준 12789 (도키도키 간식드리미) 

## 문제 링크 

[문제 링크 : https://www.acmicpc.net/problem/12789](https://www.acmicpc.net/problem/12789)

## 문제 접근

간식은 번호 순서대로 받고, 현재 번호표를 받은 사람들은 무작위로 서있습니다. 이 줄은 먼저 온 사람이 먼저 나가는 구조입니다. 즉 queue로 생각할 수 있습니다. 

두 번째로, 원래 대기열에 존재하는 사람이 임시로 대기할 수 있는 선입선출 구조를 가진 또 하나의 대기열이 추가로 존재합니다. 이는 stack으로 생각할 수 있겠네요.

따라서, queue에 있는 사람을 한 명씩 내보내며, 간식 받을 차례의 사람이라면 간식을 받고, 아니라면 임시 대기열에서 기다리는 구조로 구현을 하였습니다.

만약 대기열에 있는 사람이 다 빠져나갔음에도 불구하고, 임시 대기열에 사람이 존재한다면, "모두가 간식을 받지 못하는 순서대로 처음에 줄이 서 있었다"라고 생각할 수 있습니다.

## 소스 코드

    #include <iostream>
    #include <queue>
    #include <stack>

    using namespace std;

    int main(void){
        int n;
        cin>>n;

        stack<int> s;
        queue<int> q;

        for(int i=0;i<n;i++){
            int num;
            cin>>num;
            q.push(num); 
        }

        int order = 1;

        while(!q.empty()){
            int front = q.front(); //queue 맨 상단
            q.pop();
            
            if(front==order){ //맨 상단이 지금 들어갈 차례라면
                order++;  //넣고
                while(!s.empty() && s.top() == order) { //스택이 비지 않고, 스택 맨 상단이 들어갈 차례라면
                    s.pop();
                    order++;
                }
            }else{ //아니라면 스택(임시 대기열)에 보관함.
                s.push(front);
            }
        }

        if(s.empty()){
            cout<<"Nice";
        }else{
            cout<<"Sad";
        }

        return 0;
    }
